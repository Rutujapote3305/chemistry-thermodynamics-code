# Chemistry Thermodynamics Project

A college-level web project explaining fundamental **thermodynamics concepts** using interactive and structured HTML pages.

## Topics Covered
Laws of Thermodynamics
Thermodynamic Processes
Basic Chemistry Concepts
Educational web layout using HTML & CSS

## Technologies Used
HTML5
CSS3

## Project Structure
`chemistry1.html` – Introduction to thermodynamics
`laws.html` – Laws of thermodynamics
`process.html` – Thermodynamic processes
`style_1.css` – Styling file

## Purpose
This project was developed as part of my **college coursework** to strengthen:
Conceptual understanding of thermodynamics
Frontend basics (HTML/CSS)
Structuring educational content on the web
